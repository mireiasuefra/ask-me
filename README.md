## Getting Started ⚙️

- Clone this repo:
  ```shell
  git clone 
  ```

- Install dependencies
  ```shell
  npm install
  ```

- Run it
  ```shell
  npm start
  ```

This runs the app in the development mode. Open http://localhost:3000 to view it in the browser.
